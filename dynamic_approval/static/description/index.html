Dynamic Approval All in One

The most configurable dynamic approval app in Odoo, can be used for all default modules and custom modules

Overview

This module make configurable dynamic approval to help user add approval dynamically for all model in Odoo and this module will work both on default Odoo modules and third-party modules.

Compatibility

Community Edition

Enterprise Edition

Odoo.sh

Features

Dynamic Approval Level

Configurable approval level for each model.

Approval for All Models

Configure approval is available for model from both default odoo modules or third party modules.

Select Approval Form

Possible to apply approval only for selected form.

Easy configurations

Easy to configure approval with no need technical skill.

No need customization

One time purchase but can be used in all of your projects with no need to customization.
